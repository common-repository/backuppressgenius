== Description ==

BackupPress GENIUS sichert Ihre WordPress-Daten und lagert diese verschlüsselt auf den BackupGermany-Server aus.


= Besonderheiten =
* Automatische Sicherung zu dem von Ihnen festgelegten Zeitpunkt.
* Einfache Handhabung.
* Support Mo.-Fr. von 09:00-16:00 Uhr
* Die Daten werden mit AES (Advanced Encryption Standard) verschlüsselt.
* Die Speicherung der Daten erfolgt zudem auf bewährten Serversystemen.


== Installation ==

1. Wählen Sie „Installieren“ unterhalb des Bereichs „Plugin“ auf Ihrer WordPress-Website.
2. Klicken Sie „Plugin hochladen“; anschließend wählen Sie die .zip – Datei von BackupPress GENIUS aus.
3. Wählen Sie „Installierte Plugins“ unterhalb des Bereichs „Plugin“.
4. Klicken Sie unter „BackupPress GENIUS“ auf „Aktivieren“.
5. Gehen Sie Anschließend auf 'Einstellungen' -> 'BackupPress GENIUS', um Verschlüsselungskennwort und Zeitpunkt der automatischen Datensicherung festzulegen.

Weitere Informationen erhalten Sie auf unserer [BackupPress Support-Website](http://backuppress.de/support/)
